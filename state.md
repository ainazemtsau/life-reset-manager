# Состояние менеджера — рабочая память (ДЛЯ менеджера; владельцу — рендер по запросу)

Это durable-память. Новый чат/провайдер продолжает отсюда + спрашивает дельту «что изменилось».
Владелец сырьё не читает — ему рендеришь читаемый вид.

## Running-state
- cycle: (где мы: планирование недели / день N / review недели)
- current_method: (из recipe.md; пока НЕ ЗАДАН — см. процедуру 4.1)
- current_protections: (что сейчас под защитой — напр. восстановление активно)
- last_day_review: (дата / ссылка)

## Накопитель (ничто не теряется)
- problems: (проблемы-к-починке — копятся к review недели)
- misses: (что не сделано)
- ideas / backlog: (capture без обязательства)

## Записи (добавляются по ходу)
- weeks: (записи недель)
- days: (записи дней)
